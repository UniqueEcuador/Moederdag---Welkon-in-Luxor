# 🌙 Welkom in Luxor, Mama ✨

Een interactieve webpagina gemaakt met liefde — speciaal voor mijn moeder die nu in **Luxor, Egypte** is! 🏛️

---

## 📁 Bestanden

```
📂 repo
 ├── index.html     ← De hoofdpagina (open dit in je browser)
 ├── video.mp4      ← Jouw video (voeg deze zelf toe!)
 └── README.md      ← Dit bestand
```

---

## 🚀 Hoe te gebruiken

### Optie A — Met de video al in de map (aanbevolen)
1. Kopieer jouw MP4-bestand naar deze map
2. Hernoem het naar **`video.mp4`**
3. Open `index.html` in een browser → de video laadt automatisch!

### Optie B — Video live inladen
1. Open `index.html` in een browser
2. Klik op **"📁 Video laden"**
3. Kies je MP4-bestand → klaar!
